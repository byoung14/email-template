Mapping a Network Drive in Windows 10

Hello Customer,

We received on email inquiry about how to properly map a network drive in Windows 10. The support team here at Dunder Mifflin values customer interaction and we have provided step-by-step instructions below on how to properly map a network drive.

  1. Open File Explorer from the taskbar or the Start menu.
  2. Select This PC from the left pane, then on the Computer tab, select Map network drive.
  3. In the Drive list, select a drive letter.
  4. In the Folder box, type the path of the computer or folder you would like to use or select the Browse button to find the available computer or folder.
  5. Select Finish.

If you would prefer more visual step-by-step instruction, then here is a Youtube video presented by the channel Microsoft Lab.
